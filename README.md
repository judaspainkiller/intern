# Решение задачи ранжирования документов по релевантности. 

# О чем проект
Задание заключается в разработке модели для решения задачи ранжирования документов по релевантности.

# Инструменты
# Язык программирования: Python
Библиотеки: Pandas, Matplotlib, Numpy, Sklearn, XGBoost, Seaborn
Среда разработки: Google Colab

# Структура проекта
- загрузка датасета, просмотр первых 10 строк
- проверка на наличие пустых значений в датасете
- деление данных на обучающую и тестовую выборку
- импорт всех необходимых для работы библиотек
- создание функции для вычисления метрики ERR
- разработка модели с использованием алгоритма XGBRanker из библиотеки XGBoost
- обучение модели и вычисление метрик ndcg@5 и ERR
- вычисление дополнительных метрик MSE и accuracy
- визуализация важности признаков
- анализ метрик
  
Среднее значение метрики ndcg@5 оказалось равным 0.9. Показатели метрики ERR, однако, были довольно низкими - около 0,2.
Датасет для обучения был проанализован, была выявлена причина низкого значения метрики ERR, которая заключалась в дисбалансе классов.
В представленном датасете ранги были от 0 до 4, документов с рангом 0 было больше всего, а документов с рангами 3 и 4, что привело к низким показателям ERR.
